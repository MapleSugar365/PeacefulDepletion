## 功能
- 移除了原版和平模式自动回复饱食度和生命值的机制，并且允许了在和平、简单和普通模式下也能被饿死。另外，当安装了 `Tough As Nails（意志坚定）` 模组后，还会禁止和平模式下水分值的自动回复。
